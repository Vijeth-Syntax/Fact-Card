# Fact Card
A simple and responsive web project that displays key information in a visually appealing card format. Each card highlights a specific fact with a title, icon/image, and brief description. Ideal for showcasing quick knowledge bites, animal traits, tech specs, or educational content. Built using semantic HTML, styled with CSS Grid or Flexbox, and optionally enhanced with JavaScript for interactivity like theme toggling or filtering.

## Languages Used
- HTML
- CSS
- JS

### Features 
- It shows 10 Web Dev Facts.
- It will reset Facts once it get at last one.
